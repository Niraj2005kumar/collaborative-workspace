# Real-Time Collaborative Workspace

A full-stack Agile Project Management application developed as part of my internship at Zaalima Development Pvt. Ltd.

The application enables teams to collaborate efficiently through Kanban boards, real-time task synchronization, collaborative communication, and secure workspace management.

---

## Project Overview

This project is inspired by modern project management tools like Jira and Trello. It provides a centralized workspace where teams can create projects, manage tasks, collaborate in real time, and track project progress.

---

## Features

- User Authentication (JWT)
- Workspace Management
- Board Management
- Kanban Board
- Drag & Drop Tasks
- Real-Time Task Synchronization
- Real-Time Comments
- Typing Indicators
- Task Search
- Notifications
- Redis Caching
- Role-Based Access Control

---
